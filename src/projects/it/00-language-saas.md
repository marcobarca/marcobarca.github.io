- Architettura end-to-end di una piattaforma SaaS B2C per [LanguageBoost](https://languageboost.it/): lezioni di lingua live e interattive (fino a 100 partecipanti per classe) con registrazioni on-demand, progettata per scalare a 100k utenti e 50 sessioni live simultanee
- Design cloud-native su Azure: microservizi Spring Boot (Event, Content, User Profile), Azure Communication Services e Media Services per video live e registrazioni, login federato con Azure AD B2C, Azure SQL, Blob Storage e CDN
- Governance tecnica: definizione delle API tramite Azure API Management, sicurezza e compliance (OAuth2/OIDC, Key Vault, GDPR), roadmap MVP incrementale con percorso evolutivo verso una modalità broadcast per grandi eventi
