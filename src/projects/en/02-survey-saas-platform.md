Multi-tenant SaaS that fully automates telephone survey workflows end-to-end: outbound call orchestration, Azure Speech transcription, AI-generated reports via LangChain, a RAG conversational agent (Azure Cognitive Search) for natural-language data exploration, and native multilingual support. Serverless event-driven architecture with CI/CD and multi-tenant isolation via Azure Entra ID.
